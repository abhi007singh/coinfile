## KoinX Assignment

# Setup

1. Clone this repository

```bash
~$ git clone https://github.com/abhi007singh/coinfile.git
```

2. Copy example env to env

```bash
~$ cp .env.example .env
```

4. Install all the dependencies

```bash
~$ npm install
```

5. Run the server

```bash
~$ node .
```

6. Server on: http://localhost:3000/api/v1/trade
